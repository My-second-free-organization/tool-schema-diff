# FlowForge Documentation

Welcome to FlowForge - the enterprise business process automation platform.

## What is FlowForge?

FlowForge is a low-code/no-code platform for automating business processes. Design workflows visually, integrate with your existing systems, and let AI optimize your operations.

## Key Features
- Visual workflow designer (BPMN 2.0)
- 50+ pre-built integrations
- AI-powered task routing
- Real-time analytics
- Enterprise security (SSO, RBAC, audit logs)
